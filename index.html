Sobre Mim

Olá! Sou Lucas, desenvolvedor web especializado em criar soluções digitais eficientes e modernas. Atualmente trabalho com desenvolvimento web através da ferramenta ScriptCase.

Tenho experiência com diversas tecnologias web como HTML, CSS, JavaScript e PHP, com foco maior no PHP e banco de dados MySQL. Iniciei minha carreira com o framework CodeIgniter 3, o que me proporcionou uma base sólida em desenvolvimento web.

Além do desenvolvimento web, tenho como hobby a criação de jogos utilizando a engine Unity3D, onde desenvolvo não apenas scripts, mas também ilustrações utilizando ferramentas como Photoshop CS6 e PaintTool SAI.
